# Campaign Quality Checklist

Used by Valentina before every CEO submission. Each item is a test of honesty, not taste.

---

## Voice
- [ ] No forbidden vocabulary: "luxury", "premium", "exclusive", "iconic", "stunning", "amazing", "beautiful" (unless irreplaceable), "proud to present", "world-class", "dream home", "breathtaking", "elevated", "curated lifestyle"
- [ ] Spanish equivalents also checked: "lujoso", "exclusivo", "premium", "icónico", "impresionante", "espectacular", "sorprendente"
- [ ] Every sensory detail is traceable to the architect's intake notes in `brief.json`
- [ ] Tone is quiet, precise, and present-tense where possible
- [ ] No exclamation marks anywhere

## Completeness
- [ ] Spanish copy complete for every planned post
- [ ] English copy complete for every planned post
- [ ] English reads as written natively, not translated
- [ ] Visual plan references only assets that exist in `assets.json`
- [ ] Platform calendar has specific publish dates, not "week 1" or "TBD"
- [ ] Every post in the strategy has a corresponding copy entry and visual plan entry

## Brand fit
- [ ] Project name used correctly in all copy (never generic description)
- [ ] Location named specifically (never "a home in Baja" — always "Cerritos, Baja California Sur")
- [ ] Collaborators and builders credited where the architect's intake includes them
- [ ] Content communicates inhabitation, not objecthood — the reader is placed inside the space
- [ ] CTA present only in Priority 5 (conversion) campaigns — absent in atmospheric posts

## Bilingual integrity
- [ ] Spanish written first, English derived from meaning not words
- [ ] Instagram format correct: ES → blank line → — → EN → blank line → hashtags
- [ ] YouTube format correct: ES title, ES description, blank line, —, EN description
- [ ] Spanish architectural terms left in Spanish when English equivalent is weaker

## CEO package
- [ ] `approval-package.json` generated and complete
- [ ] Approval page HTML generated and locally served
- [ ] All assets referenced in the visual plan are accessible
- [ ] WhatsApp message draft prepared with correct approval URL
- [ ] Valentina's campaign summary paragraph written (her voice, not Diego's)

---

## Valentina's Final Question

Before sending to CEO, ask yourself:

*"Does this content sound like something the architects would say, or does it sound like something a marketing team would say?"*

If it sounds like marketing, it goes back to Diego. It does not go to the CEO.
