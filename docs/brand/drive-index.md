## How to use Drive IDs in the intake process

When Arquitecto runs intake, reference the Drive folder directly in assets.json:

```json
{
  "drive_folder_id": "FOLDER_ID_HERE",
  "drive_folder_url": "https://drive.google.com/drive/folders/FOLDER_ID_HERE",
  "photos": "see Drive folder — request specific filenames from architects",
  "video_files": "see Drive folder"
}
```

Ileana can navigate the Drive folder to identify available assets by folder ID when the architects provide specific filenames.
