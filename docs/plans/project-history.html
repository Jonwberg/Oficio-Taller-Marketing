Scope template evolution
2016 — One-off scopes per project (ALCANCE_[NAME].docx). Each project had unique scope language. Owned by oficiotaller account.
2017–2020 — Continuation of per-project scopes. Account transitions from oficiotaller → andrea.martinezr. Scopes archived Dec 2020 during cleanup.
Jan 2021 — ALCANCE_BASE.docx committed to 01.8 ALCANCE BASE folder. Standardized template established. Also appears in 00.2 FORMATOS (format library). Multiple copies = being refined.
Feb 2021 — ALCANCE_BASE moved to official 01.8 ALCANCE BASE folder under andrea.martinezr account.
2021–2023 — All new projects use 01.4 ALCANCE or 01.5 ALCANCE folder containing a derived ALCANCE_[PROJECT].docx from the base. New account: andrea. Also: info account active on parallel projects. valeria joins Jun 2023.
2024+ — Entrega pipeline now defines ALCANCE_BASE as SOW source template feeding into scope-of-work.json generation.
