# Press Kit Build Plan — Casa Atlas

## Structure of each deliverable

Each kit is a single self-contained HTML file containing:
1. **Email** — styled email preview with From / To / Subject / body pitch
2. **Project narrative** — adapted from Marcela's text per category
3. **Image gallery** — curated photo selection per category
4. **Architectural documentation** — plans/elevations (A1/A2 mandatory; B referenced; C omitted)
