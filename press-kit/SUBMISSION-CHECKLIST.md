# Cross-Publication Submission Checklist

Quick reference for preparing press kits across all target publications.

## Image Requirements

| Publication | Min. Images | Resolution | Drawings Required |
|---|---|---|---|
| ArchDaily | 15 | 2880px+ width | **Yes** (plans, sections) |
| Yellowtrace | 10-15 | High-res | No |
| The Local Project | 10-30 | Professional | Optional |
| Architecture Hunter | 5-7 (max 20) | 5MB max/photo | No |
| AD Mexico | 10-15 | 300 DPI (print) | No |
| Architectural Digest | 10-15 | 300 DPI (print) | No |
| Yinji Space | 10-15 | High-res | Optional |

## Content Requirements

| Publication | Narrative Style | Word Count | Language |
|---|---|---|---|
| ArchDaily | Technical case study | 300-500 | English/Spanish |
| Yellowtrace | Sensorial, personal | 600-1200 | English |
| The Local Project | Process-focused | 500-1000 | English |
| Architecture Hunter | Minimal/evocative | 50-100 | English |
| AD Mexico | Cultural storytelling | 1000-2000 | Spanish |
| Architectural Digest | Editorial narrative | 2000-3000 | English |
| Yinji Space | Poetic, atmospheric | 200-500 | Chinese/English |

## Exclusivity & Timing

## Submission Order Strategy

Given exclusivity requirements, submit in this order:

1. **Architectural Digest** (US) — first, requires exclusivity, 7-month process
2. **AD Mexico** — after AD US decision, prefers exclusivity
3. **All others** — can be submitted simultaneously after exclusive windows close

## Key Reminders

- Never send the same press kit to everyone
- Each publication gets a **distinct narrative angle**
- Photography is the universal entry ticket — invest in a great photographer
- ArchDaily is the only one that **requires** architectural drawings
- The Local Project only covers AU/NZ/North America
- AD (US) sends their own photographer if accepted — initial photos are for evaluation only
