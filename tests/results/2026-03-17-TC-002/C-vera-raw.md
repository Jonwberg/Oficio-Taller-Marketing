# Segment C — Vera (Cost Basis)

Vera computed the preliminary cost basis using a MXN 24,000/sqm benchmark for residential_in_development in Los Cabos, BCS. Base construction cost: 337sqm × MXN 24,000 = MXN 8,088,000. Architecture fee at 12% = MXN 970,560, consistent with the seed data USD 68,000 total fee. Total estimate including engineering allowance and 10% contingency: MXN 10,230,840. Special features (pool, solar, smart home) absorbed within contingency at this preliminary stage.
